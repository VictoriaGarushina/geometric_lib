# Math formulas
## Square
### These formulas are for a square, finding its area and perimeter
- Area: *S = a²*
- Perimeter: *P = 4a*

## Circle
### These formulas are for a circle, finding its area and perimeter
- Area: *S = πR²*
- Perimeter: *P = 2πR*

## Rectangle
### These formulas are for a rectangle, finding its area and perimeter
- Area: *S = ab*
- Perimeter: *P = 2(a + b)*
